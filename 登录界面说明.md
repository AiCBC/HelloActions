# 登录界面应用

## 概述
这是一个基于Qt QML开发的现代化登录界面应用，具有美观的视觉设计和完整的交互功能。

## 界面特色

### 🎨 视觉设计
- **渐变背景**：蓝紫色渐变背景，营造现代感
- **圆角设计**：输入框和按钮采用圆角设计，符合Material Design风格
- **半透明效果**：界面元素具有适当的透明度，层次分明
- **阴影效果**：登录按钮具有阴影效果，增强立体感

### 🔐 功能特色
- **用户名输入**：带有用户图标的输入框
- **密码输入**：带有锁图标的密码输入框，自动隐藏密码
- **记住密码**：可选择记住密码功能
- **忘记密码**：忘记密码链接
- **登录验证**：基本的输入验证功能
- **注册链接**：新用户注册跳转

### ⌨️ 交互功能
- **键盘支持**：支持回车键快速登录
- **鼠标交互**：完整的鼠标点击和悬停效果
- **状态反馈**：实时显示登录状态信息
- **表单验证**：检查用户名和密码是否为空

## 技术规格

### 开发技术
- **框架**：Qt 5.15.15 + QML
- **语言**：C++ + QML
- **UI库**：Qt Quick Controls 2
- **图形效果**：Qt Graphical Effects

### 界面尺寸
- **窗口大小**：400 x 600 像素
- **输入框高度**：50 像素
- **按钮高度**：50 像素
- **界面适配**：支持不同屏幕尺寸自适应

## 使用方法

### 运行应用
```bash
# 方法1：使用启动脚本
./run_login.sh

# 方法2：直接运行
./bin/release/HelloActions-Qt
```

### 编译项目
```bash
# 生成Makefile
qmake HelloActions-Qt.pro

# 编译项目
make
```

### 系统要求
- **操作系统**：Linux (Ubuntu 18.04+)
- **Qt版本**：Qt 5.15+
- **图形环境**：需要X11或Wayland图形环境
- **显存**：建议512MB以上

## 界面预览

### 主要组件
1. **Logo区域**：圆形背景的锁图标
2. **标题**："欢迎登录"文字
3. **输入区域**：用户名和密码输入框
4. **选项区域**：记住密码复选框和忘记密码链接
5. **操作区域**：登录按钮
6. **注册区域**：新用户注册链接

### 颜色方案
- **主背景**：#667eea 到 #764ba2 渐变
- **输入框**：白色半透明 (opacity: 0.9)
- **按钮颜色**：#FF9800 (按下时: #FFA726)
- **文字颜色**：白色
- **成功状态**：#4CAF50
- **错误状态**：#ff5722
